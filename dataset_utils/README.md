# Dataset Preparation Artifacts

This directory preserves intermediate files from the thesis dataset-cleaning workflow.

The polished splits used by the training, inference, and evaluation scripts live in `data/`. Files here document the provenance of those splits: cleaned table metadata, generated query lists, dismissed IDs, MySQL notes, and large context/query exports used while constructing the final Text-to-SQL dataset.

These artifacts are intentionally retained for research transparency. They are not required for the normal fine-tuning or evaluation commands in the root README.
